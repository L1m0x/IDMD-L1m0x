# Changelog

All major changes to the project **Instagram Direct Message Deleter** will be recorded here.

## [1.2.0] - 2026-03-24
### Added
- New function **Logout/Switch Account** (Option 7).
- Display message content (Live Text) when you delete.
- Support for Load More on Inbox for viewing all conversations.

## [1.1.0] - 2026-03-22
### Added
- **Full Scan 5000 messages**.
- Identification and appearance of Usernames instead of dried IDs.
- Separation of our own messages vs messages from another user.

## [1.0.0] - 2026-03-20
### Added
- Initial version of the script (Unsend core logic).
- Session (JSON) Login system.
- Basic UI with ANSI colors.
